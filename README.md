# react-native-portal-animation

React Native library for creating animated portals, enabling smooth transitions and shared element animations between components or screens.

## Installation


```sh
npm install react-native-portal-animation
```


## Usage


```js
import { multiply } from 'react-native-portal-animation';

// ...

const result = await multiply(3, 7);
```
